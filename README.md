# Email Fraud Detection & Visualization System

## 📌 Project Description
This project focuses on detecting and analyzing fraudulent (spam) and legitimate (ham) emails using Machine Learning, Natural Language Processing (NLP), and data visualization techniques.

## 🎯 Objectives
- Analyze spam vs ham emails  
- Visualize important patterns in email data  
- Build ML models for spam detection  

## ⚙️ Tech Stack
- Python (Jupyter Notebook)
- Pandas, NumPy
- Scikit-learn
- NLTK
- Matplotlib, Plotly
- WordCloud

## 📊 Features
- Data cleaning and preprocessing  
- NLP-based text processing  
- TF-IDF & CountVectorizer  
- Spam classification  
- Data visualization (charts & word clouds)  
- Model training and evaluation  

## 📊 Dataset
This project uses an email dataset containing spam and ham messages.

## 📈 Visualizations
- Pie Chart (Spam vs Ham)  
- Bar Charts (Top words)  
- Histograms (word & character count)  
- WordCloud  

## 🔍 Data Preprocessing
- Removed duplicates  
- Handled missing values  
- Cleaned text using NLTK  
- Created new features (word & character count)  

## 🧪 Model Building
- TF-IDF & CountVectorizer  
- Logistic Regression, Naive Bayes, XGBoost  
- Accuracy evaluation  

## ▶️ How to Run
1. Clone this repository  
2. Install dependencies:
3. Open `analysis.ipynb` in Jupyter Notebook  
4. Run all cells
